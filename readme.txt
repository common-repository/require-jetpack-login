= Warning =

Before activating this plugin, make sure that Jetpack plugin is installed and you connect your WP.Com account with your site. If yes, then please activate it to use "WordPress.com log in". To do so, just follow the steps below:
1. Go to Jetpack > Settings
2. Click Security
3. In the "WordPress.com log in" section, enable "Allow users to login this site using WordPress.com Account"

== Installation ==

If you are on WordPress website then: 

1.  Go to **Plugins** and click on **Add new**.
2.  Search for Plugin "**Jetpack Login by Harman Singh Hira**" in the directory.
3.  Now, **Install** and **Activate** it.

Or, if you want to download it in your machine and want to upload it via FTP then do according to following below steps:

1.  Download Plugin in your machine
2.  Extract the zipped file and upload the folder \`ut-insert-title\` to \`/wp-content/plugins/\` directory.
3.  Activate the plugin through the \`Plugins\` menu in WordPress.
